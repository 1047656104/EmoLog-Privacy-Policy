EmotiCare 隐私策略

保护您的隐私是我们的首要任务

生效日期：2025年1月25日

最后更新：2025年1月25日

1. 引言

欢迎使用EmotiCare应用（以下简称"本应用"或"EmotiCare"）。我们深知个人信息对您的重要性，并会尽力保护您的个人信息安全可靠。我们致力于维持您对我们的信任，恪守以下原则，保护您的个人信息：权责一致原则、目的明确原则、选择同意原则、最少够用原则、确保安全原则、主体参与原则、公开透明原则等。

本隐私策略将帮助您了解以下内容：

  • 我们如何收集和使用您的个人信息
  • 我们如何存储和保护您的个人信息
  • 您如何管理您的个人信息

2. 我们收集的信息

2.1 情绪与健康数据（核心功能）

作为一款专注于情绪健康管理的应用，我们会收集以下健康相关信息：

  • 情绪健康评估数据：您完成的情绪健康评估问卷、焦虑抑郁量表、压力水平测试、睡眠质量评估、社交能力评估等专业测评结果
  • 情绪记录数据：您记录的每日心情状态、情绪描述、情绪触发事件、心情变化趋势等情绪健康信息
  • 情绪健康计划：基于评估结果生成的个性化情绪改善计划、任务完成状态、进度跟踪、情绪健康目标设定等
  • 压力管理记录：您使用的放松练习、正念冥想、呼吸训练等压力缓解活动记录
  • 健康行为数据：睡眠质量记录、运动活动、社交互动、自我关怀活动等影响情绪健康的行为数据
  • 关联情绪图片：您选择上传的与心情记录相关的图片，用于增强情绪表达和记录完整性

2.2 应用使用数据

为了提供更好的健康管理服务，我们会收集：

  • 签到记录：您的每日健康签到状态、连续使用天数等，用于建立健康习惯
  • 互动数据：您的点赞、分享等积极互动行为，用于提升情绪健康体验
  • 个性化设置：您的健康提醒偏好、主题选择、通知设置等个性化配置
  • 数据导出记录：当您选择导出健康数据时的相关操作信息

2.3 技术支持信息

为了确保应用稳定运行和改进健康服务质量：

  • 设备信息：设备型号、操作系统版本等基本设备信息
  • 应用使用统计：功能使用频率、会话时长等匿名化统计信息
  • 错误日志：应用崩溃或错误信息，用于改进应用稳定性和用户体验

3. 信息使用目的

我们收集和使用您的个人信息专门用于提供情绪健康管理服务，具体目的包括：

3.1 核心情绪健康管理功能

  • 情绪健康评估：提供专业的情绪健康测评工具，包括焦虑、抑郁、压力、睡眠质量等多维度评估，生成个性化情绪健康报告
  • 情绪记录与分析：帮助您记录和追踪日常情绪变化，识别情绪模式和触发因素，提供情绪健康洞察
  • 个性化情绪健康计划：基于评估结果制定专属的情绪健康改善计划，包括压力管理、情绪调节、睡眠改善等任务
  • 压力管理支持：提供正念冥想、呼吸练习、放松技巧等科学的压力缓解方法和指导
  • 情绪健康趋势分析：分析您的心情变化趋势、情绪健康行为模式，提供可视化的情绪健康统计报告

3.2 情绪健康习惯培养

  • 情绪健康签到系统：通过每日签到机制帮助您建立持续的情绪健康管理习惯，提供连续使用激励
  • 积极情绪互动促进：通过点赞、分享等积极互动功能，营造支持性的情绪健康环境
  • 个性化情绪提醒：根据您的情绪健康目标和偏好，提供个性化的情绪健康提醒和建议

3.3 服务优化与支持

  • 情绪健康数据管理：提供安全的数据导出功能，让您完全掌控自己的情绪健康信息
  • 个性化情绪体验：根据您的使用习惯和情绪健康需求，优化界面和功能体验
  • 情绪健康服务改进：基于匿名化的使用统计，持续改进情绪健康服务质量
  • 技术支持：解决技术问题，确保情绪健康管理服务的稳定性和可靠性
  • 安全保障：保护您的情绪健康隐私数据，防止未经授权的访问和使用

4. 情绪健康信息存储和保护

4.1 情绪健康数据本地存储

考虑到情绪健康数据的敏感性，EmotiCare采用严格的本地存储策略：

  • 完全本地化：您的所有情绪健康评估结果、情绪记录、情绪健康计划等敏感数据都存储在您的设备本地
  • 零云端传输：我们承诺不会将任何情绪健康相关数据上传到服务器或云端存储
  • 设备隔离：每个设备的情绪健康数据完全独立，不会与其他设备或用户共享

4.2 敏感情绪健康数据保护措施

针对情绪健康数据的特殊性，我们实施以下专门保护措施：

  • 数据加密：所有情绪评估数据和情绪记录在本地存储时进行加密处理
  • 访问控制：仅应用本身可以访问情绪健康数据，其他应用无法获取
  • 安全删除：当您选择删除数据时，系统会安全彻底地清除相关信息
  • 匿名化处理：用于改进服务的统计数据经过完全匿名化处理，无法追溯到个人

4.3 情绪健康隐私保护承诺

  • 医疗级隐私标准：我们参照医疗健康行业的隐私保护标准来处理您的情绪健康数据
  • 最小化原则：只收集提供情绪健康服务必需的最少数据
  • 透明化处理：所有情绪健康数据的使用都在本策略中明确说明，绝不用于其他目的
  • 用户完全控制：您可以随时查看、修改、导出或删除所有情绪健康数据

5. 情绪健康信息共享政策

鉴于情绪健康数据的高度敏感性，我们实施最严格的信息共享限制：

零共享承诺：我们承诺绝不会向任何第三方（包括医疗机构、保险公司、雇主等）出售、出租、共享或以其他方式披露您的情绪健康数据。

5.1 情绪健康数据保护原则

  • 完全保密：您的情绪评估结果、情绪记录等情绪健康数据仅供您个人使用
  • 无商业用途：情绪健康数据不会用于任何商业目的或营销活动
  • 无研究用途：不会将您的情绪健康数据用于学术研究或数据分析
  • 无第三方访问：任何第三方服务商都无法访问您的情绪健康信息

5.2 极端情况处理

仅在以下极端法律情况下，我们可能被迫提供有限信息：

  • 法院强制令或法律明确要求
  • 涉及重大公共安全威胁的紧急情况
  • 保护用户或他人生命安全的紧急情况

即使在上述情况下，我们也会尽最大努力保护您的情绪健康隐私，仅提供法律要求的最少必要信息。

6. 您的权利

关于您的个人信息，您拥有以下权利：

  • 访问权：您可以随时查看您的所有心情记录、计划数据、签到记录、评估结果等
  • 更正权：您可以修改或更正您的心情记录、计划内容、个人设置等
  • 删除权：您可以删除单条记录、清除特定类型数据或清除所有数据
  • 导出权：您可以将所有数据导出为JSON格式进行备份
  • 管理权：您可以管理计划任务、调整签到设置、修改评估信息等
  • 撤回同意：您可以随时停止使用应用或删除应用及其所有数据

7. 第三方服务

EmotiCare是一个完全离线的应用，不集成任何第三方服务或SDK。我们不会向第三方服务提供商传输您的任何数据。

8. 儿童隐私保护

我们非常重视儿童的隐私保护。EmotiCare不会主动收集13岁以下儿童的个人信息。如果您是13岁以下的儿童，请在家长或监护人的指导下使用本应用。

9. 隐私策略更新

我们可能会不时更新本隐私策略。当我们对隐私策略进行重大变更时，我们会通过应用内通知或其他适当方式通知您。请您定期查看本隐私策略以了解最新信息。

继续使用EmotiCare即表示您同意接受更新后的隐私策略。
